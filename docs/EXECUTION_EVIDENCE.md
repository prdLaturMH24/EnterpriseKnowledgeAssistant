# Execution Evidence Checklist

Capture and submit at least 5 screenshots (recommended 6–7):

1. **Application Startup**
   - Terminal showing `python main.py` launch and mode selection prompt.

2. **Agent Execution Workflow**
   - CrewAI verbose execution showing planner/research/mcp/response/evaluation activity.

3. **RAG Retrieval Process**
   - Output section `--- Retrieved Context (RAG) ---` with source/page evidence tags.

4. **MCP Server Interaction**
   - Output section `--- MCP Supplemental Data ---` showing filesystem-derived updates.

5. **RAGAS Evaluation Results**
   - Output section `--- RAGAS Evaluation ---` with all four mandatory metrics and interpretation.

6. **Final Output**
   - Output section `--- Final Answer ---`.

7. **Execution Summary (Optional but recommended)**
   - Output section `=== Execution Summary ===`.

## Suggested Demo Queries
- What is the leave policy for full-time employees?
- How should employees report a security incident?
- What are the escalation steps for critical support tickets?

## Submission Notes
- Ensure screenshots are clear and readable.
- Remove any secrets before capture.
- Include screenshots together with source code and documentation package.
